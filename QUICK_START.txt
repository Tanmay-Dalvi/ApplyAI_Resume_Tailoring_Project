========================================
  AI RESUME TAILOR - CHROME EXTENSION
========================================

QUICK START
-----------

1. Build the extension:
   npm run build

2. Load in Chrome:
   - Open chrome://extensions/
   - Enable "Developer mode"
   - Click "Load unpacked"
   - Select the "dist" folder

3. Use the extension:
   - Go to any LinkedIn job posting
   - Click extension icon
   - Upload PDF resume → Extract Text
   - Click "Scrape Current LinkedIn Job"
   - Click "Generate Tailored Resume"
   - DOCX file downloads automatically!
